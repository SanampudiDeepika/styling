CASCADING STYLE SHEETS

CSS

CSS stands for Cascading style sheets.The version we use for css is version-3.Cascading Style Sheets, is a style sheet language used to describe the visual presentation of a document written in HTML. CSS describes how elements should be rendered on screen. It enables web designers to control the layout, colors, fonts, and other visual aspects of multiple web pages all at once. In Cascading Style Sheets (CSS), cascading is the logic that browsers use to determine which CSS rulesets are most important when they conflict with each other. The cascade is central to CSS and is used to solve conflicts when multiple CSS rules apply to an HTML element.


SYNTAX: propertyname:value



How to insert CSS:

Inline Styling:

In inline styling CSS styles directly apply to the individual HTML elements using the style attribute.


SYNTAX: <tag style = " "></tag>
Internal Styling:

Internal styling in CSS refers to the method of embedding CSS rules directly within the HTML document using the <style> tag.


SYNTAX: <style> /* Internal CSS starts here */ <style>
External Styling:

External CSS is used to style multiple HTML pages with a single style sheet. External CSS contains a separate CSS file with a .css extension.


SYNTAX: <link rel="stylesheet" href=mystyle.css>



Simple Selectors:

Tag:

SYNTAX: p { border: 5px blue; margin: 15px; }


Class:

SYNTAX: .class { css declarations; }
Id:

SYNTAX: #change1 { text-align: center; color: orange; }
Group:

Grouping selectors in CSS allow you to apply the same styles to multiple selectors.



SYNTAX: h1,h2,p { text-align: center; color: blue; }


Universal:

SYNTAX:* { color: green; font-size: 20px; }
